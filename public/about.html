About Is It Critical?

This web app uses the NVD API but is not endorsed or certified by the NVD

What is the NVD?

The National Vulnerability Database (NVD) is the U.S. government repository of standards-based vulnerability management data maintained by NIST. It provides detailed information on publicly disclosed cybersecurity vulnerabilities.

What is a CVE?

A CVE (Common Vulnerabilities and Exposures) is a publicly disclosed security vulnerability or exposure. Each CVE is assigned a unique ID (ex. CVE-2026-1234) and contains information about the vulnerability, its severity, and affected products.

What is CVSS?

The Common Vulnerability Scoring System (CVSS) is a standardized way of rating the severity of CVEs on a scale from 0 to 10. Scores are broken into four severity levels:

  • Critical: 9.0 to 10.0
  • High: 7.0 to 8.9
  • Medium: 4.0 to 6.9
  • Low: 0.1 to 3.9

What does Is It Critical? do?

Is It Critical? utilizes raw CVE data from the NVD to create a more user-friendly way of exploring it. It allows you to navigate recent CVEs, filter by severity or keyword, visualizes some trends, and drill down into specific vulnerabilities.

Disclaimer

This web app uses the NVD API but is not endorsed or certified by the NVD or NIST. Data may not reflect the most current vulnerability information. Always refer to the official NVD website at nvd.nist.gov for reliable information.
